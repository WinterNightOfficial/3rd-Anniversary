不知不覺夜雪羽也已經成立了三年
至今大家仍在，實在是得來不易
儘管寮友各自可能互不認識
但大家總在群聊時不時閒談、說笑幾句
看起來距離也不再是那麼地遙遠
我很高興能夠有幸成立了夜雪羽
讓我、大家都認識到了不少的新朋友
來年，也請夜雪羽的寮友們多多指教了
夜雪羽、三週年快樂！
